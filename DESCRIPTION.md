# Heavenly Convergence Compatibility Patch

Compatibility patch for Incandescent Nexus and the existing Heavenly Convergence patch.

This mod is a patch-of-a-patch: it does not replace either dependency. Load it after both so Heavenly Convergence Art is updated to match the current Ascend From Nine Mountains condensation art data.

## Features

- Updates Heavenly Convergence Art to use the current Core Formation condensation art data.
- Combines the current Mortal Condensation, Cycle Of Nine, Eternal Immolation, and Distilling Furnace benefits.
- Collapses the Cycle Of Nine, Eternal Immolation, and Distilling Furnace combat effects into one Heavenly Convergence buff.
- Renames the droplet-consumption resonance effect to Convergent Resonance.
- Removes obsolete condensation-art cost fields from the patched item data.

## Requirements

- Incandescent Nexus
- The existing Incandescent Nexus patch

## Load Order

1. Incandescent Nexus
2. Incandescent Nexus patch
3. Heavenly Convergence Compatibility Patch

Verified against an installed AFNM runtime reporting v0.6.65-2026-06-26-nightly.
